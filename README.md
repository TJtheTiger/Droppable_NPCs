# Droppable NPCs

Hades II mod allowing you to get every NPC in every route. Has support for Zag's Journey.

## Features

- Surface Droppable NPCs: Arachne, Narcissus, Echo, Sisyphus, Eurydice, and Patroclus
- Underworld Droppable NPCs: Medea, Circe, Icarus, Sisyphus, Eurydice, and Patroclus
- Nightmare(Zag's Journey Route)  Droppable NPCs: Arachne, Narcissus, Echo, Medea, Circe, and Icarus
